# Use of CSS Layout to recreate octocats.

The following layout tools were studied and applied to applicable areas on the project.
1. Box model
  * padding-box
  * border-box
2. Position
  * fixed
  * relative
  * absolute
3. Floats
  * left
  * right
  * none
  * clear
4. clear fix hack using overflow: auto;
